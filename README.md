# taaabs-base-graoh
`<taaabs-base-graph>` is a graphical element that displays the graph of a ktbs base.

Example:

    <taaabs-base-graph id="tbg" taaabs-central-loading base-url="[[YOUR_BASE_URL]]" language="[[language]]"></taaabs-base-graph>
